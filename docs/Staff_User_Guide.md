# Staff User Guide – OCSS GT Lobby Check-In

## Introduction

This guide is for **OCSS front-desk staff** who use the Lobby Check-In system to manage the student queue during office hours.

---

## Accessing the Application

Open a web browser and navigate to the URL provided by your IT administrator (e.g., `https://ocss-lobby.gatech.edu`). The application will load the **Kiosk Check-In** page by default.

To access staff pages, click **"Staff Login"** in the sidebar and enter your username and password.

---

## Staff Queue (`2_Staff_Queue`)

### What you'll see
The queue page displays all students who have checked in today and are currently **Waiting** or **In-Progress**.

| Column | Description |
|--------|-------------|
| Student Name | First and last name |
| GT ID | Student's Georgia Tech ID |
| Appointment Time | Scheduled time |
| Appointment Type | Type of appointment |
| Counselor | Assigned staff member |
| Wait Time | Minutes since check-in |
| Status | Waiting / In-Progress / Completed |

### Actions
- **Mark In-Progress**: Click the button next to a student's name when they are called into the office. This updates their status to `In-Progress`.
- **Mark Completed**: Click when the appointment is finished. This records the check-out time.
- **Refresh**: The queue auto-refreshes every 30 seconds. Click **Refresh Now** to update immediately.

---

## No-Show Finalization (`4_NoShow_Finalization`)

At the end of the day (or after a scheduled appointment window has passed):

1. Navigate to the **No-Show Finalization** page.
2. Select the appointment date using the date picker.
3. The page will list all appointments for that date that **do not have a check-in record**.
4. Review the list and click **Finalize No-Shows** to mark them as `NoShow` in the system.
5. A confirmation message will display the number of records updated.

> **Note:** This action cannot be undone without contacting an administrator.

---

## Logging Out

Click **Logout** in the sidebar to end your session. The application will return to the Kiosk Check-In page.

---

## Frequently Asked Questions

**Q: A student says they checked in but they are not in the queue.**
A: Ask the student to confirm their GT ID and try checking in again at the kiosk. If the problem persists, contact an administrator to verify the appointment record exists.

**Q: The queue is not updating.**
A: Click the **Refresh Now** button. If the page is unresponsive, refresh your browser.

**Q: I accidentally marked a student as Completed.**
A: Contact an administrator to correct the record via the Admin page.
